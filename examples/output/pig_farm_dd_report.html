LevelDescription
1Narrative / anecdotal only (self-reported, no data)
2Output data (quantified activities, e.g. # served, # trained)
3Outcome data measured (pre/post, surveys, tracked KPIs)
4Controlled comparison (quasi-experimental, benchmarks, cohort studies)
