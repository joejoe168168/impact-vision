# LinkedIn post — Impact Vision v0.14.0 (launch)

## Option A — 180 words, founder-voice, most exciting v0.1 → v0.14 features

---

Two years ago I pushed a 400-line Python script that could parse the GIIN IRIS+ Excel and nothing else. Today Impact Vision v0.14.0 ships.

One command turns a pitch deck into an entire impact workflow for a VC / PE / impact fund:

→ **Screen**  ·  5-Dimension score, SDG alignment, greenwashing flag in under a minute.
→ **DD**  ·  122-question risk-first checklist auto-exports a founder questionnaire in HTML *and* Word.
→ **IC**  ·  Deal-gate engine with fund thesis, PASS / WARN / FAIL verdict, and a branded investment memo.
→ **Monitor**  ·  Bayesian evidence updater, meta-analysis against J-PAL / 3ie, spillover + SROI adjustments, satellite & worker-voice connectors.
→ **Report**  ·  Hash-chained LP feed, ILPA portal, ISAE 3000 / CSRD / ISSB / SOC 2 packs, 7-jurisdiction regulatory disclosures.

All reachable through CLI · REST · MCP · Streamlit · a brand-new **Web Console** that auto-generates a form for every endpoint.

787 IRIS+ metrics · 17 SDGs · 10 ESG frameworks · 26 tools · 150 / 150 tests green · MIT license.

Built on OpenHarness. Feedback very welcome.
